Оператор CREATE DATABASE Описать базу данных, состоящую из четырех файловых групп, одного журнала. 
В первичной группе описать 2 файла, разбросанные по различным дискам (один из них имеет размер 200 МБ, рост до 300 МБ, с шагом 20 МБ). 
Три вторичные группы: каждая состоит из одного файла (в первой группе файл растет с шагом 10 МБ, размер не ограничен, а во второй имеет постоянный размер).
Вторая группа является группой по умолчанию. Файл журнала растет на 15%.

Оператор ALTER DATEBASE. Изменить базу данных, удалив вторую и третью вторичные группы, а по умолчанию сделать первичную группу. Добавить два файла в журнал.
